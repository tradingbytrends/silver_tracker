 How to Run

  1. Install dependencies
  pip install -r requirements.txt

  2. Set up credentials
  cp .env.example .env
  # Edit .env with your Angel One API key, TOTP secret, and
  Twilio credentials

  3. Run
  python main.py

  What happens at startup

  1. Validates all required env vars — fails fast with a
  clear error if missing
  2. Logs into Angel One SmartAPI (with TOTP 2FA)
  3. Backfills the last 7 days of silver minute candles into
   SQLite
  4. Starts two scheduled jobs:
    - price_job — every 60 seconds, fetches the latest 1-min
   candle and upserts it
    - notify_job — every 15–20 minutes (randomized each
  interval), builds a summary and sends WhatsApp

  WhatsApp message format

  *Silver Price Update*
  Price: ₹87,450.00
  Time : 2026-04-21 14:35 IST
  Change: ▲ ₹120.00 (+0.14%)
  1H Range: ₹87,200.00 – ₹87,600.00
  Day Range: ₹86,900.00 – ₹87,800.00

  To get your Angel One SILVER token

  Check SILVER_TOKEN in .env.example — it's pre-set to
  234230 (MCX Silver). Verify it on the
  https://margincalculator.angelbroking.com/OpenAPI_File/fil
  es/OpenAPIScripMaster.json if needed.
